# **App Name**: Calculador de Megas

## Core Features:

- User Account Management: Secure user registration with email, phone number, and password confirmation. Includes functionality for user login and a password recovery process.
- Data Profit Calculator: An intuitive form allowing users to input the volume and cost of purchased data (e.g., GB) and the desired selling price per unit, then instantly display the calculated profit.
- Personalized Calculation History: A user-specific view showing a log of all previously performed profit calculations for quick reference and tracking.
- AI Profit Advisor Tool: Provides suggestions for optimal selling prices per GB based on the user's purchasing costs and desired profit margins, acting as a smart tool for maximizing earnings.

## Style Guidelines:

- Primary color: A balanced, professional blue (#3973AD), representing clarity and trustworthiness. Derived from an analogous color scheme emphasizing efficiency.
- Background color: A very light, desaturated blue (#ECF0F3), maintaining a clean and clear canvas suitable for a light scheme.
- Accent color: A vibrant, analogous teal-cyan (#4BCECE), used to highlight calls to action and draw attention to important metrics, suggesting growth and innovation.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, neutral, and highly readable characteristics, suitable for displaying numerical data and forms clearly.
- Use a set of simple, crisp line icons that relate to data, finance, and calculations, ensuring visual consistency and ease of understanding.
- Adopt a clean, spacious layout with clear input fields and prominent display areas for results. Prioritize responsive design for optimal viewing across various devices.
- Implement subtle micro-animations for user feedback upon successful calculation or form submission, providing a smooth and engaging user experience without distraction.
